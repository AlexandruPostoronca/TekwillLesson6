Equal arrays

Given four arrays of integers:

int[] numbers1 = { 1, 2, 3, 4 };
int[] numbers2 = { 1, 2, 3, 4 };
int[] numbers3 = { 4, 3, 2, 1 };
int[] numbers4 = { 1, 2, 3 };

Select all invocations that return true.

Arrays.equals compares 2 arrays and returns true if they contain the same elements in the same order, false - otherwise
